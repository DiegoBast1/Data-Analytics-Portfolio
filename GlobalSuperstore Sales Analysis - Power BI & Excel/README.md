# GlobalSuperstore Sales Analysis with Power BI and Excel

## Project Purpose
Clean data and analyze Total Sales by Year, Average Profit by Market, Average Profit by Category and Total Sales.

## Data sources
- GlobalSuperstore csv file

## Tools Used
- Excel
- Power BI to load, clean, transform and create visuals with data to answer business questions.

## Key Insights Found
- Technology is the most profitable category driving 41% of total profit while Furniture represents 30%
- Unites States, APAC and the EU are the most profitable markets
- Amount: 93% growth from 2011 to 2015; generating $2.19M in Total Sales
- highlighted opportunity to improve profitability by prioritizing technology products.
